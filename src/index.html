Portafolio - Proyectos

Diseño Web

Páginas web atractivas y funcionales con el objetivo de satisfacer todas las expectativas del cliente. La importancia del diseño y la funcionalidad es fundamental para tener éxito y atraer a los usuarios. También lo es la velocidad de carga del sitio web, deben ser rápidos para que el visitante tenga la mejor experiencia de usuraio.

Responsive Design

Hoy en día las páginas web se ven en una gran variedad de dispositivos como tablets, celulares, libros electrónicos, notebooks, PCs, etc. La tecnología móvil es el presente y el futuro. Por eso es importante asegurarnos de que el sitio web sea compatible con todos los tamaños de pantalla que existen en la actualidad.

SEO

Al crear sitios web, hay que tener muy en cuenta la optimización de los motores de búsqueda. Hay que llevar a cabo prácticas de SEO estándar en cada página del sitio web, lo que permite que el sitio tenga una clasificación más alta en los resultados de Google y asi generar mas tráfico o visitas a nuestra página web.

Cipher

Aplicación para la generación y descarga de sellos digitales y facturación electrónica.

Los usuarios son los Contribuyentes (Personas físicas y morales), que pagan impuestos y facturan sus productos y/o servicios.

Con esta aplicación y por medio de un despacho de contadores, obtener ante la SHCP la autorización para la generación de sellos digitales que incluye la facturación electrónica.

Para el desarrollo de proyecto se hizo uso de Html5, CSS3 y Vanilla JavaScript.

Data Lover

Aplicación para el manejo de datos, visualización de gráficas y tablas de información.

Los usuarios pueden visualizar la data de Personas heridas por medio de transporte en EEUU, seleccionando el tipo de transporte y el periodo. La aplicación les mostrara como resultado de su solicitud, la data representada en una Gráfica de Pie 3D y una lineal, así como la tabla correspondiente a los datos.

Para el desarrollo del proyecto se hizo uso de HTML5, CSS3, JavaScript y Google Charts.

Social Network

La aplicación es una Red Social pensada en los amantes del código. Donde pueden "Crear, Leer, Actualizar y/o Borrar" sus Post o imagenes.

La idea del proyecto "CodDev", fue con la finalidad de crear un espacio donde la comunidad Tech pueda compartir información y conocimiento.

Para el desarrolo de la aplicación si hizo uso de HTML5, CSS3, JavaScript, Firebase, Firestore Database, Materialize. Es un proyecto colaborativo por lo que usamos Trello para planeación y Github para el control de versiones de código.
